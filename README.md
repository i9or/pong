# Pong

Self-explanatory, I guess.

## Setup

Clone the repository

```sh
git clone --recursive https://github.com/i9or/pong.git
```

and open it in any C++ IDE with CMake support. I use CLion, by the way.

## Run

Click "Run" button in your IDE.
